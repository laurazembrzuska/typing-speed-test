# typing-speed-test
## Description
"Typing Speed Test" is a web application to measure typing speed and accuracy. Users can assess their typing skills by typing a given text within a user-selected time frame. The user can choose between a Polish/English interface. Times to choose from: 60, 120, 180 seconds. He can stop, continue and reset the test.

## How to Run the Application

To run this application locally, follow these steps:

1. Clone the repository to your local machine using the following command:

   ```
   git clone https://github.com/laurazembrzuska/typing-speed-test.git
   ```

2. Navigate to the project directory:

   ```
   cd typing-speed-test
   ```

3. Install the necessary dependencies by running the following command:

   ```
   npm install
   ```

4. Once the dependencies are installed, start the development server using the following command:

   ```
   npm start
   ```

5. Your default web browser should automatically open and display the application. If it doesn't, you can manually open your browser and navigate to [http://localhost:3000](http://localhost:3000).

6. You can now use the "Typing Speed Test" application to measure your typing speed and accuracy.
